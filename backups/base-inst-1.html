General Instructions

This HIT involves performing a number of short, similar tasks: you will view a stream of letters, and respond to each one.
Each task has a different TARGET sequence, and you should respond by pressing the "F" key (the Target Key), whenever the current letter matches that target sequence.
Whenever the current letter does not match the target sequence, you should respond by pressing the "J" key (the Non-Target Key). Explicit instructions will be provided before each task.
